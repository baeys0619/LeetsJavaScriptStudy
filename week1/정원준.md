# [3주차] 한 입 크기로 잘라먹는 리액트 학습 정리

## 📘 학습 범위
- 자바스크립트 기초 학습(섹션2)
## 📘 새로 알게된 점


이번 섹션에서는 자바스크립트의 기본 문법을 다루면서 여러 가지를 새로 배웠다.  
먼저 **템플릿 리터럴(Template Literal)** 문법을 익혔다.  
백틱(`` ` ``)을 사용하면 문자열 안에서 `${변수명}` 형태로 손쉽게 변수를 삽입할 수 있고, 여러 줄 문자열도 표현할 수 있다.  
예를 들어 `let introduceText = \`${myName}은 ${myLocation}에 삽니다.\`;` 와 같이 작성하면 문자열 결합이 훨씬 간결해진다.

또한 **null**과 **undefined**의 차이도 배웠다.  
`null`은 의도적으로 “비어 있음”을 표현하는 값이고,  
`undefined`는 변수가 선언되었지만 아직 값이 할당되지 않은 상태를 의미한다.  
이와 함께 **null 병합 연산자(??)** 도 배웠는데, 이는 왼쪽 값이 `null`이나 `undefined`일 때만 오른쪽 값을 사용하는 연산자다.  
예를 들어 `let name = null; let user = name ?? "Guest";` 라고 하면 `user`에는 `"Guest"`가 들어간다.

함수의 선언 방식도 다시 정리했다.  
함수 선언문(`function funcA(){}`)은 **호이스팅**되어 어디서든 호출할 수 있지만,  
함수 표현식(`const funcB = function(){}`)과 **화살표 함수**(`const funcC = () => {}`)는 호이스팅되지 않는다.  

또한 **콜백 함수(callback function)** 개념도 다뤘다.  
콜백 함수란 함수를 인자로 전달해서, 나중에 실행되도록 하는 함수다.  
예를 들어 `repeat(5, i => console.log(i * 2));` 처럼 사용하면, 반복문 안에서 전달된 함수를 반복적으로 호출할 수 있다.

마지막으로 **객체의 속성 유무를 확인하는 방법**으로 `in` 연산자를 배웠다.  
예를 들어 `“name” in person`은 해당 객체에 `"name"` 속성이 존재하면 `true`, 아니면 `false`를 반환한다.  
이 연산자를 사용하면 객체 구조를 확인할 때 유용하다.

이번 학습을 통해 자바스크립트 기본 개념과 문법이 한층 더 명확해졌다.
